## Run Locally

To run Save-Your-Tasks locally, run this command on your git bash:

Linux and macOS:
```bash
sudo git clone https://github.com/MaciejMi/Save-Your-Tasks.git
```

Windows:
```bash
git clone https://github.com/MaciejMi/Save-Your-Tasks.git
```
